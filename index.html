ABOUT ME

Get to know me!

Saya adalah mahasiswa Jurusan Teknologi Informasi, Fakultas Ilmu Komputer, Universitas Jember angkatan 2024. Saat ini saya sedang menempuh pendidikan dengan minat yang besar pada bidang web development. Ketertarikan saya pada dunia teknologi, khususnya pengembangan web, berawal dari rasa ingin tahu bagaimana sebuah situs dapat dibangun dan memberikan pengalaman yang bermanfaat bagi penggunanya. Hal tersebut mendorong saya untuk terus belajar dan mendalami keterampilan teknis dalam merancang, mengembangkan, dan mengoptimalkan aplikasi berbasis web.

Fokus utama saya adalah mempelajari dasar-dasar pengembangan front-end dan back-end. Dari sisi front-end, saya tertarik untuk memahami bagaimana membangun tampilan antarmuka yang tidak hanya menarik secara visual, tetapi juga responsif dan mudah digunakan. Sementara itu, dari sisi back-end, saya ingin memahami bagaimana sistem bekerja di balik layar, termasuk bagaimana data dikelola dan diintegrasikan untuk mendukung fungsionalitas website.

PROJECTS

  • Aplikasi PlantTani

    • PlantTani adalah aplikasi yang dibangun menggunakan bahasa pemrograman Python dengan tujuan membantu petani dalam mencatat dan mengelola data pertanian. Sistem ini mendukung pencatatan lahan, tanaman, kebutuhan pupuk, pestisida, hingga peralatan pertanian. Data disimpan dalam format CSV sehingga tetap sederhana namun rapi dan transparan. Fitur utama PlantTani adalah pencatatan transaksi operasional pertanian serta rekapitulasi agar petani dapat memantau aktivitas mereka secara lebih teratur.

    Melalui project ini, saya membuktikan keterampilan dalam menggunakan Python untuk membangun aplikasi berbasis data. Saya berhasil menerapkan manajemen file CSV, logika bisnis sederhana, serta struktur program yang jelas..

  • Aplikasi Pengelolaan Sistem Manajemen Panti Asuhan "Asuh.in"
    • "Asuh.in" adalah aplikasi desktop yang dibangun menggunakan C# dengan basis data PostgreSQL. Aplikasi ini dirancang untuk menjawab permasalahan nyata di panti asuhan yang masih mengelola data secara manual, seperti kesalahan pencatatan, kehilangan arsip, serta kesulitan dalam membuat laporan. Melalui sistem digital yang terstruktur, “Asuh.in” mampu meningkatkan efisiensi, transparansi, dan akuntabilitas dalam pengelolaan data anak asuh, donatur, inventaris, kegiatan, serta keuangan sederhana.

    Dalam pengembangan project ini, saya mengimplementasikan kemampuan dalam merancang aplikasi berbasis Object Oriented Programming (OOP), menyusun modul untuk berbagai jenis pengguna (admin, pengurus panti, bendahara, dan donatur), serta mengintegrasikan sistem dengan database relasional.
